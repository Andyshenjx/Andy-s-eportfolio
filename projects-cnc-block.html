Project

CNC Block

CNC programming, coordinate setup, machining operations, and engraving.

Goal

Goal

  • Learn the basics of CNC programming and CNC machining.
  • Practice setting machine coordinates and tool length.
  • Create a finished CNC wooden block with a custom engraved design.

Tools Used

Tools Used

  • CNC Milling Machine
  • Different size of drills
  • Conversational Programming Software

What I Did

What I Did

  • Mounted and referenced the wooden stock.
  • Set the X, Y, and Z coordinates for the machine.
  • Completed facing, profile, and pocket operations.
  • Changed tools for the engraving operation.
  • Engraved a Conan image/profile onto the CNC block.

Safety

Safety

  • Made sure the wood block was clamped securely before machining.
  • Don’t open it when its running
  • Checked the setup before starting each operation.

Skills & Reflection

Skills & Reflection

  • I learned how to set CNC coordinates and tool length accurately.
  • I gained experience using different CNC operations, including facing, profiling, pocketing, and engraving.
  • I learned how to turn a design into a machined and engraved final product.
  • This project helped me understand the importance of precision, setup, and safety in CNC machining.
