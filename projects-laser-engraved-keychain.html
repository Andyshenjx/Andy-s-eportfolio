Project

Laser Engraved Keychain

Manual graphing, coordinates, LightBurn programming, and laser engraving.

Goal

Goal

  • Learn how to create a custom laser-engraved keychain.
  • Practice manual graphing and coordinate programming.
  • Use LightBurn to transfer a design into a finished wooden keychain.

Tools Used

Tools Used

  • Laser Engraver
  • LightBurn Software
  • Graph Paper
  • Pencil
  • Protective Tinted Glasses

What I Did

What I Did

  • Designed my name on graph paper.
  • Worked out the X and Y coordinates needed for the design.
  • Wrote the code manually on the graph paper.
  • Transferred the code into a Google Doc.
  • Programmed the design in LightBurn.
  • Used the laser engraver to engrave my name onto a wooden keychain.

Safety

Safety

  • Wore protective tinted glasses while operating the laser.
  • Made sure not to look directly at the laser.
  • Stayed aware of people around the laser while it was operating.

Skills & Reflection

Skills & Reflection

  • I learned how to use coordinates to create a laser engraving design.
  • I practiced manual coding, graphing, and transferring coordinates into software.
  • I gained experience using LightBurn and the laser engraver.
