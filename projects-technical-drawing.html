Project

Technical Drawing

Orthographic views, scale, measurement, and clean technical drawing.

Goal

Goal

  • Learn how to draw a tool in three orthographic views.
  • Keep the scale and size consistent in each view.
  • Practice accurate measurement and clean technical drawing.

Tools Used

Tools Used

  • Pencil
  • Ruler
  • Tape measure
  • Graph paper

What I Did

What I Did

  • Selected a tool and drew it from three different views.
  • Measured the tool carefully before drawing.
  • Used a scale so the drawing could fit on the paper.

Skills & Reflection

Skills & Reflection

  • I learned how to measure an object and turn it into a scaled technical drawing.
  • I practiced using a ruler and tape measure accurately.
  • I also learned that drawing lightly at first is helpful because it makes corrections easier.
