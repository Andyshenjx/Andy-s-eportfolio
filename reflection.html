Favorite Project: CNC Block

Why I Liked It

I like this project because it allows me to freely display the characters I like on the CNC blocks.

Process

  1. Mount the stock securely in the CNC machine.
  2. Release the E-stop, reset the machine, and set the limit switches.
  3. Set the X, Y, and Z zero points.
  4. Perform the facing operation to create a flat surface.
  5. Create the profile of the block using conversational programming.
  6. Machine the pocket feature to the required dimensions.
  7. Import and adjust the engraving design by scaling and rotating it.
  8. Run the engraving operation and inspect the finished CNC block.

Problem I Had

I wanted to engrave an anime character image, but I only had a paper sketch of the image. The machine required a DXF file for engraving, and the result was very poor when I directly converted the file format.

How I Fixed It

I used an iPad and GoodNotes to draw the main outline of the character and exported it as a JPG file. I then used a converter to convert the JPG file into an SVG vector graphic. After that, I imported it into AutoCAD for modification and finally exported it as a DXF file. By observing the tool path movement, I used AutoCAD again to make more precise adjustments to the machining path.

What I Learned

  • How to set up and operate a CNC machine.
  • I learned facing, profiling, pocketing, and engraving.
  • How to use conversational programming to create machining operations.

What I Learned

  • How to convert image files into DXF files for CNC engraving.
  • How to use AutoCAD to modify and improve tool paths.
  • The importance of testing, troubleshooting, and making adjustments to achieve better machining results.
